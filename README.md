## Description

Blog application, powered by [Nest](https://github.com/nestjs/nest), based on MySQL & Redis.

Have a quick look on: [ifuyun.com](https://www.ifuyun.com)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
